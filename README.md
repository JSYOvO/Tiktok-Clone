# Tiktok Clone
I ran a Tiktok clone with React. 

### Used
    FRONTEND
        React
        Firebase hosting
        Material UI
        Flexbox
        3rd party ticker

    BACKEND
        Node.js
        Express.js server
        Mongodb
        Mongoose
        Hosting on Heroku

    HOSTING
        Firebase(Firestore Database)
